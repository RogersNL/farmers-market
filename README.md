# Avery's Farmer's Market Schedule
##### A Farmer's Market App

## Created 09/05/2018

## Made By

  * Nick Rogers  
  * Swati Sahay

## Description

A site that shows the schedule and produce available for a farm.

## Setup

  * Git clone the file  
  * Run npm install  
  * npm start

## Technologies Used

  **ReactJS**
